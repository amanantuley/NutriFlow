# 📖 Overview

**NutriFlow** is a modern AI-powered nutrition platform designed to simplify healthy living. It enables users to plan meals, monitor nutrition, track calories, and receive personalized dietary recommendations based on their goals and preferences.

Built with modern web technologies, NutriFlow delivers a fast, responsive, and scalable user experience suitable for fitness enthusiasts, health-conscious individuals, and anyone looking to improve their diet.

# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/amanantuley/NutriFlow.git
```

### Navigate to the Project

```bash
cd NutriFlow
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Visit:

```text
http://localhost:3000
```
