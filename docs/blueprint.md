# **App Name**: FlavorFind

## Core Features:

- Restaurant Discovery: Browse restaurants by cuisine, location, and dietary preferences.
- Menu and Order Customization: View detailed menus, customize food orders, and specify dietary restrictions.
- Shopping Cart and Checkout: Manage a multi-item shopping cart, apply discounts, and securely complete checkout.
- Order Tracking: Real-time order tracking and delivery updates.
- Personalized Recommendations: AI-powered food recommendations based on past orders and preferences; uses past user behavior as a tool to inform selections

## Style Guidelines:

- Primary color: Fresh green (#6AB04A), reflecting the freshness of ingredients and food.
- Background color: Clean white (#F5F5F5), to emphasize cleanliness and readability.
- Accent color: Yellow-green (#AEEA00), for interactive elements, buttons, and highlights to create a vibrant interface.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif.
- Use minimalist and recognizable icons to represent different cuisines and food categories.
- Clean and intuitive layout with clear hierarchy for easy navigation and browsing.
- Subtle animations for transitions, loading states, and user feedback.
